# JD-GPT
### SQL Intelligence System v1.0

## Features

- Plain English → SQL conversion
- Multi-dialect support
- PostgreSQL
- MySQL
- SQL Server
- HiveQL
- Snowflake
- BigQuery
- Production-ready SQL
- Dual AI architecture
- Automatic failover
- Assumptions and explanations

## Dual AI Architecture

🔵 JD Core
Primary Intelligence Engine

🟣 KD Core
Adaptive Backup Intelligence

⚡ Automatic Failover Enabled

git clone https://github.com/jagadish710/jd-gpt.git

cd jd-gpt
